# Foursquare-Gospel-Church-Oworo-1
The Foursquare Gospel Church is a global evangelical Pentecostal denomination founded in 1923 by Aimee Semple McPherson in Los Angeles, California. It focuses on preaching Jesus Christ as Savior, Baptizer (with the Holy Spirit), Healer, and Soon-Coming King. The church is known for its emphasis on evangelism,, with millions of members worldwide.
